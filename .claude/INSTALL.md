# Installing HA NOVA for Claude Code

## Quick Start

```bash
git clone https://github.com/markusleben/ha-nova.git ~/ha-nova
cd ~/ha-nova && npm install
npx ha-nova setup
```

The wizard handles everything: App installation, authentication, and skill setup.

## Already Set Up?

Run diagnostics:

```bash
npx ha-nova doctor
```
